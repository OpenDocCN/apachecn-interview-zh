## 简介[](#introduction "Direct link to heading")

动态规划通常用于解决优化问题。提高 DP 的唯一方法就是练习。认识到一个问题可以通过 DP 来解决需要一定的实践。

## 技巧[](#techniques "Direct link to heading")

有时，您不需要将整个 DP 表存储在内存中，矩阵的最后两个值或最后两行就足够了。
